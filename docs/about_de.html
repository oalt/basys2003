über BASys 2003.

überblick.

BASys 2003 ist ein in Java implementiertes Softwaresystem zur Planung und Inbetriebnahme von Gebäudeautomationen. Das System entstand im Rahmen der Diplomarbeit von Oliver Alt an der TU Wien (Österreich). Es ermöglicht die Planung einer Gebäudeautomation, die Inbetriebnahme der Automation und bietet Funktionen im laufenden Betrieb den Gebäudezustand zu überwachen und zu beeinflussen.

anforderungen.

Das System entstand unter folgenden Anforderungen:

  • Offene erweiterbare Datenformate
  • Unterstützung verschiedener Bussysteme der Gebäudeautomation
  • Einfache, intuitive Bedienbarkeit
  • Plattformunabhängigkeit

eigenschaften.

Die aktuelle Version des Systems besitzt folgende Eigenschaften:

  • Planung einer Gebäudeautomation per Drag and Drop in einer strukturellen Ansicht.
  • Intuitive Planung durch automatische Aktor-/Sensorsuche für ein zu steuerndes Gerät.
  • Unterstützung des Europäischen Installations Bus (EIB). EIB Installationen sind plan- und programmierbar.
  • Plattformunabhängigkeit duch ausschließliche Implementierung in Java.
  • Zukunftssichere Datenhaltung im XML Format.
  • Fehlersuchfunktionen für den EIB (Debugging auf Busebene).
  • Import und administration der EIB Herstellerdaten für die systemeigene Datenbasis.

zukünftige erweiterungen.

Für zukünftige Versionen des Systems sind folgende Erweiterungen geplant:

  • Einbindung von CAD-Daten bei der Planung. Dies ermöglicht noch intuitiveres Planen, Installieren und erleichtert die Fehlersuche vor Ort.
  • Versionskontrolle der Projektdaten und Unterstützung von Mehrbenutzerarbeit an einem Projekt.
  • Anbindung von weiteren Gebäudebussystemen (z.B. LONWorks, LCN,...).
  • Einbindung anderer Medien in die Planung (z.B. Steuerung von Hifi-Komponenten). BASys 2003 könnte hierbei eine Gatewayfunktionalität übernehmen.

Wenn Sie weitere Ideen für die Weiterentwicklung haben, schreiben Sie uns unter ideen@basys2003.org
