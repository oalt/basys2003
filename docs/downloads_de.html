zum betrieb notwendige daten.

Zum korrekten Betrieb von BASys 2003 sind Daten notwendig, welche die Informationen über die anzusprechenden Busgeräte enthalten.

eib-devices.xml (9kB) Einfache Beispieldatenbank mit Aktoren und Sensoren. Bitte ins Verzeichnis ${BASYS_HOME}/global-data/eib/ kopieren.
produktdaten.zip (6MB) XML Version der Produktdaten der Firmen ABB und Somfy. Bitte entpacken und ins Verzeichnis ${BASYS_HOME}/global-data/eib/devicedata/ kopieren.

Das ${BASYS_HOME} Verzeichnis variiert je nach Betriebssystem. Auf Unix/Linux Systemen ist es /home/[logninName]/basys/. Auf Windows 2000 basierten Systemen ist es C:\Documents and Settings\[loginName]\basys\. Dieses Verzeichnis wird beim erstmaligen Start von BASys 2003 automatisch angelegt!

java web start.

Sie können die BASys 2003 Applikation mit Hilfe von Java Web Start direkt mit unten stehendem Link starten. Dazu ist ein installiertes Java Runtime Environment (JRE) Version >=1.4 erforderlich. Das JRE finden sie bei http://www.javasoft.com zum download.
